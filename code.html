Problem Statement:

Write a code in any programming language that converts Centigrade to Fahrenheit.

The problem statement is to write a code in any programming langauge to covert the given Centigrade temperature into Fahrenheit. The code is written in C langauge. Go through the sample input and output. Variables used in the program are given below. The complexity of the program is mentioned below.

Code:

            int main()  
 
        {  
 
                float fahrenheit, celsius;
   
                printf("Temperature in Celsius: ");

                scanf("%4f", &celsius);

                fahrenheit =( (celsius*9)/5)+32;
  
                printf("\n\n Temperature in fahrenheit is:  %f",fahrenheit);  

                return (0); 
 
    	}

celsius, fahrenheit
